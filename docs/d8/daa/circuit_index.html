Circuit Definition

This paper contains a description of the following PLONK-style circuits:

  • Poseidon Circuit (for state Merkle-tree and proof generation).
  • SHA2-256 Circuit and SHA2-512 Circuit (for block headers and transactions hashes verification)
  • Merkle Tree Circuit
  • Ed25519 Circuit (for validators signature verification)
  • Correct Validator Set Proof Circuit
